Roll Billing Codes

Non-Billable Item Year End Processing
This Feature Can Only be Run Mid-December to January End

Selecting 'Add New Non-Billable Codes' will:
  • Create the new 'PTO' code and add it to all active salaried employees required to submit timesheets.
  • Create the new 'Bench' code and add it to all active salaried employees required to submit timesheets that currently have a bench code assigned to them.
  • Create the new 'Holiday' code which is added to salaried employee timesheets based on the holiday schedule of their assigned client(s) and/or projects(s).
Selecting 'Inactivate Non-Billable Codes' will:
  • Inactive all 'PTO', 'Bench', and 'Holiday' codes for the year indicated.
